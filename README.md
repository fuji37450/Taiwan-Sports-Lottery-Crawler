# 台灣運彩爬蟲
每分鐘紀錄美國職棒（MLB）和美國職籃（NBA）於台灣運彩「勝分差」玩法的賠率，並匯出成 Excel 檔。

- 每場比賽為檔案中的一個工作頁
- 當賠率有更新時以紅字標示
- 比賽結束後記錄最終比分
